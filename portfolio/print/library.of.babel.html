Overview

This is an illustrated book design for “Library of Babel” written by Jorge Luis Borges.

  • Typeface

      This book is considered as a guidance handbook for THE LIBRARY OF BABEL. In this book, you could find the main architecture, the floorplan diagram, the identity and the direction to find an item in the library.
